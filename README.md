## ğŸ› ï¸ Installation & Setup

<details>
<summary><b>ğŸš€ Quick Start Guide</b></summary>

### Prerequisites
- Node.js (v18+ recommended)
- Docker & Docker Compose
- Git
- PostgreSQL (v13+)

### Installation Steps

```bash
# Clone the repository
git clone https://github.com/Anujmishra2005/PM_Accelerator_task.git
cd PM_Accelerator_task

# Install dependencies for Tech Assessments
cd Tech_Assessments
npm install
npm run build

# Install dependencies for Project Solutions
cd ../Project_Solutions
npm install
npm run build

# Setup environment variables
cp .env.example .env
# Configure your environment variables

# Start development servers
npm run dev:all

# Or run with Docker
docker-compose up -d
```

</details>
